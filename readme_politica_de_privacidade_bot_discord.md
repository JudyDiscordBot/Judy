# Política de Privacidade

## 1. Coleta de Informações
O bot pode coletar informações mínimas necessárias para seu funcionamento, como:
- ID de usuários, servidores e canais.
- Conteúdo de comandos enviados diretamente ao bot.

Nenhuma informação sensível é coletada intencionalmente.

## 2. Uso das Informações
As informações coletadas são utilizadas exclusivamente para:
- Executar as funcionalidades do bot.
- Melhorar desempenho, estabilidade e experiência do usuário.
- Garantir segurança e prevenir abusos.

## 3. Armazenamento de Dados
Os dados podem ser armazenados de forma temporária ou permanente, dependendo da funcionalidade. Medidas razoáveis de segurança são aplicadas para proteger essas informações.

## 4. Compartilhamento de Dados
Nenhuma informação coletada é vendida ou compartilhada com terceiros, exceto quando exigido por lei ou para cumprimento de obrigações legais.

## 5. Retenção e Exclusão
Os dados são mantidos apenas pelo tempo necessário para a finalidade proposta. O usuário pode solicitar a exclusão de dados, quando aplicável, através do canal de suporte do bot.

## 6. Privacidade de Menores
O bot não é destinado a menores de 13 anos. O uso deve respeitar as regras de idade mínima do Discord.

## 7. Alterações nesta Política
Esta Política de Privacidade pode ser modificada a qualquer momento. Alterações entram em vigor imediatamente após a publicação.

## 8. Contato
Em caso de dúvidas sobre esta Política de Privacidade ou sobre dados coletados, entre em contato pelos meios oficiais disponibilizados pelo bot.
